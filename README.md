## Building This Website Locally

### Windows

#### Prerequisites

- [Ruby+DevKit](https://rubyinstaller.org/downloads/)

#### Terminal Commands

```powershell
# Install Jekyll
gem install jekyll

# Install Jekyll Feed
gem install jekyll-feed

# Check if Jekyll is properly installed
jekyll -v

# Serve Jekyll site
jekyll serve

# Build Jekyll site locally (not needed as GitHub will build on push)
jekyll build
```
